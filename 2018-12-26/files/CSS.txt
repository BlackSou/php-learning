1. Подключение CSS:
 -> <link href='style.css' rel='stylesheet' type='text/css'>
 	все стили собраны в одном месте, кешируется, отдельный файл
 -> <style> div { color: green; } </style>
 	все стили собраны в одном месте, не кешируется, тот же файл
 -> <div style="font-weight: bold;"> "инлайновые" стили
    стили разбросаны по проекту, не кешируется, тот же файл

ПРИОРИТЕТЫ!!!

2. АНТИ команда - аттрибут important
{ color: red !important; }

3. АНТИ команда - @import
<style> @import url('a.css'); </style>

4. Селекторы
 -> "*" - универсальная маска, применить ко всем
 -> "<имя_тега>" - применится к тегу, явное указание
 -> "<имя_класса>" - применится к классу (в css через точку ".")
 -> "<имя_id>" - применится к id (в css через хеш "#")
 -> multiple - комбинированные, составные, например тег+класс:
   => тег1, класс1, тег2 - группировка через запятую
   => тег1 тег2 - вложенные - через пробел (между тег1 и тег2 могут быть другие теги!!!)
   => тег1 > тег2 - "дочерние" - через знак "больше" (между тег1 и тег2 НЕ могут быть другие теги!!!)
   => тег1 + тег2 - "сестринские" - через знак "+" (два соседних элемента на одном уровне)
   => label[for="name"], input[type=text] - Селекторы атрибутов тега 
   => ПСЕВДОКЛАССЫ - правила состояния элемента или событий (:hover | :visited) - через знак ":" после
