Шпаргалка по консольным командам:

1. Пути:
 - "относительный" => путь к файлу/папке относительно данной директории,
   например "../sites/myProject"
 - "абсолютный" => путь к файлу/папке начиная от корне диска/хранилища,
   например "/users/vasya/home/sites/myProject"
 - "." -> означает текущую папку
 - ".." -> директория на уровень выше
 - "~" -> домашняя директория пользователя (в Линукс)

2. Хелперы:
 - "pwd" - "print working directory" (показать абсолютный путь "курсора")
 - "ls" - "list" (показать список файлов и папок в текущей диретории)
   флаги:
     => "a" - "all" показть скрытые и системные файлы и папки
     => "l" - "long format" показать свойства файлов/папок
     => "t" - "time modify order by" сортировать по времени изменения
   пример использования: "ls -at"
 - "cd" - "change directory" изменить директорию
   пример исользования: "cd ~/sites/myProject"
 - "clear" - очистить экран терминала (набор команд остается в истории но на экране появляется свободное место)
 - "history" - показать историю команд ранее набранных в терминале
   фильтрация: "grep" - фильтр по маске, например:
   пример использования: 'history | grep "php"' - показать все команды содержащие текст php в себе
     из истории команд терминала

3. Создание:
 - "touch" - создание файла
   пример использования:
     "touch file.ext" - создать файл в текущей директории с названием file и расширением ext
     "touch ../sites/index.php" - создать файл в директории на уровень выше
       и поддиректории sites с названием index и расширением php
 - "mkdir" - "make directory" создание директории
   пример использования:
     "mkdir myProject" - создать папку в текущей директории с названием myProject
     "mkdir ~/sites/myProject" - создать папку в подпапке sites домашней директории с названием myProject

4. Копирование:
 - "cp" - "copy" копирование файлов/папок
   пример использования:
     "cp index.php /var/www/html/myProject/" - скопировать файл в текущей директории index.php
       в папку /var/www/html/myProject/
     "cp index.php file.ext some.txt myProject/" - скопировать файлы в текущей директории index.php, file.ext,
       some.txt (может бытьбольше трех файлов указано) в папку myProject/ находящуюся в текущей директории
     "cp * ~/sites/" - скопировать все (маска "*") файлы/папки из текущей директории
       в папку sites/ находящуюся в домашней директории
     "cp *.txt sites/" - скопировать все файлы с расширением txt из текущей директории
       в папку sites/ находящуюся в текущей директории
     "mv index.php indexNew.php" - сопировать файл в текущей директории index.php в
       файл indexNew.php в текущей директории (если indexNew.php существует то будет перезапись)

5. Перемещение (иногда переименование):
 - "mv" - "move" перемещение (иногда переименование) файлов/папок
   пример использования:
     "mv index.php /var/www/html/myProject/" - переместить файл в текущей директории index.php
       в папку /var/www/html/myProject/
     "mv index.php file.ext some.txt myProject/" - переместить файлы в текущей директории index.php, file.ext,
       some.txt (может бытьбольше трех файлов указано) в папку myProject/ находящуюся в текущей директории
     "mv * ~/sites/" - переместить все (маска "*") файлы/папки из текущей директории
       в папку sites/ находящуюся в домашней директории
     "mv *.txt sites/" - переместить все файлы с расширением txt из текущей директории
       в папку sites/ находящуюся в текущей директории
     "mv index.php indexNew.php" - !! переименовать !! файл в текущей директории index.php на
       indexNew.php в текущей директории (если indexNew.php существует то будет перезапись)

6. Удаление:
 - "rm" - "remove" удаление файлов/папок
   флаги:
     => "r" - "recursive" рекурсивная обработка подкаталогов, нужен для удаления папок
     => "f" - "force" удаление файлов/папок если директория в которой они находятся имеет защиту записи
     => "i" - "interactive" перед каждым удалением выводить запрос - подтверждение
   пример использования:
     "rm /var/www/html/myProject/index.php" - удалить файл index.php находящийся в папке
       /var/www/html/myProject/
     "rm index.php file.ext some.txt" - удалить файлы в текущей директории index.php, file.ext, some.txt
     "rm -r ~/sites/" - удалить папку sites находящуюся в домашней папке
     "mv -rf /var/www/html/sites/" - удалить папку sites находящуюся
       в папке защищенной на запись по адресу /var/www/html/
