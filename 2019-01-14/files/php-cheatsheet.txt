3. Запуск из консоли
 - php -r "echo 'hello world';" - запуск PHP скрипта из командной строки
 - php -i - Выводит информацию о текущей конфигурации PHP 
 - php --ini - Получить список конфигурационных файлов

4. Теги 
 - <?php ... ?>
 - НЕ ИСПОЛЬЗУЙТЕ: <? ... ?> - сокращенная запись (short open tags)
 - НЕ ИСПОЛЬЗУЙТЕ: <% ... %> - теги в стиле ASP.NET (ASP tags)
 - НЕ ИСПОЛЬЗУЙТЕ: <script language='PHP'> ... </script> - скриптовые теги (script tags) 

5. функции вывода на экран

6. Команды (инструкции):
 - операторы, функции, выражения - команды

7. комментарии
 - "//" - комментирование одной строки, всё что справа
 - "/* ... */" - комментирование всего что внутри

8. Вывод ошибок включить:
 - nginx включить вывод ошибок - "sudo subl /etc/php/7.1/fpm/pool.d/www.conf", параметр php_flag[display_errors] = on, "sudo service php7.1-fpm restart"
 - apache "error_reporting(E_ALL);" каждый раз писать
 - trigger_error("Cannot divide by zero", E_WARNING);
 - E_ERROR, E_NOTICE, E_WARNING и т.д.

9. Типичные ошибки
 - ";"
 - неправильные или незакрытые теги, 
 - смешение тегов HTML+PHP+HTML+PHP+HTML...
 - неправильные комментарии (незакрытые перепутанные, теги внутри комментов)
 - Обращение к не объявленной переменной

10. константы, глобальный доступ

11. Переменные и их именование
 - суть, инициализация, присвоение
 - "$" - идентификатор переменной 
 - case-sensitive
 - "_" - зарезервированный символ начала перемнной
 - начало только с "_" или (a-z)
 - допустимые символы (0-9), (a-z), (A-Z), "_" 

12. Переменные (продолжение)
 - $$abc; - переменная переменной
 - & - передача по ссылке


13. Типы данных:
	Скалярные
	 - boolean - логическое (true, false)
		 = правда/ложь
		 = true|false
		 = TRUE|FALSE
		 = 0|1
	 - int - integer, целое число
	     = 0, -5, 10
	 - float - число с плавающей точкой (дробное число)
	     = 0.5, -12.000009
	 - string - строка
	     = ", ' - двойная / одинарная кавычки
	     = " - с переменными
	     = ' - "as is" вывод как есть
	Смешанные
	 - array - массив (набор элементов)
	 - object - объект
	 - callable - с PHP7 Функции обратного вызова
	 - iterable - с PHP7 Итерируемые
	Специальные
	 - resource - ресурс
	 - null - перемення без значения, единственно возможное значение типа null
	     = состояние перменной
	     = не определена (не инициализирована)
	     = установлена пустой
	     = удалена

13. Типы данных, приведение типов 
 - явно и неявно (особенности динамической типизации в PHP)
 	= (string) $abc - явно
 	= 12 + '13' - неявно
 - расмотреть приведение каждого к каждому простому типу!!!
    = bool
    = string
    = int
    = float
    = null
 - проверка типов: is_int, is_bool
 - gettype, settype

14. Функции проверки содержимого:

15. Выражения, операторы 
 - арифметические
 	= унарные (один операнд):
 		оператор смены знака
 	= бинарные (два операнда):
 		оператор сложения: +
		оператор вычитания: -
		оператор умножения: *
		оператор деления: /
		оператор остатка от деления: %
	= приоритет 
 - инкремент/декремент (разбираем разницу в постинкременте и преинкременте)
 	= инкремент
 		- преинкремент ++$i;
		- постинкремент $i++;
		- $a += 1;
 	= декремент
 - конкатенация - склеивание строк "."
